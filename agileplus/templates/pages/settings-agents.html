Agent Configuration Help

Agent Pool Size
Number of agents that can run concurrently. Higher values allow more parallel work but consume more resources.
Retry Budget
Maximum number of times a failed task can be retried. Set to 0 to disable retries.
Dispatch Mode
Strategy for assigning tasks to agents:
  • Round Robin: Distribute tasks evenly across all agents
  • Priority: Assign to agents based on task priority
  • Balanced: Load-aware distribution based on agent capacity
  • Manual: Require explicit agent assignment for each task
