================================================================================
IF SOMETHING BREAKS
================================================================================

Build Fails:
  $ cargo build --workspace 2>&1 | head -20
  $ git diff HEAD  # review last change
  $ git checkout -- <files>  # revert if needed

Tests Fail:
  $ RUST_BACKTRACE=1 cargo test --workspace -- --nocapture
  $ git stash && cargo test && git stash pop  # check if pre-existing
  $ Fix or report as blocker

Clippy Warnings:
  $ cargo clippy --workspace -- -D warnings 2>&1 | head -10
  $ Fix with code or add justified #[allow(...)] comment

For detailed procedures: See DEPENDENCY_PHASE2_VALIDATION.md → Failure Recovery
